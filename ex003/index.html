Símbolos e Emojis no seu Site!

Vamos adicionar alguns símbolos especiais:
® -- Marca registrada
© -- Copyright
™ -- TM no alto - sinceramente n sei direito o que isso significa
€ -- Símbolo da Moeda Euro
£ -- Símbolo da Libra Esterlina
¥ -- Símbolo do Iene Japonês
¢ -- Sei lá o que é isso, outro símbolo ai
δ -- Delta Minusculo
Δ -- Delta Maiscul, o legal é que só muda a primeira letra
↑ -- Seta para cima
↑ -- Forma simplificada da Seta para cima
Outros simbolos estão descritos no PDF

Agora vamos adicionar alguns emojis:
Para isso, poderíamos só copiar e colar da web - consultando o EmojiPedia, por exemplo
O grande problema é que isso não é compatível com todos os navegadores, então não é
a abordagem mais recomendada
Daí, o mais interessante é usar o código UTF-8 do emoji. Para apresentado, usamos o formato:

& # x código_UTF-8

(tem que colocar tudo junto, separei para que o navegador não entenda como um caractere especial)
(note que é um código em hexadecimal)

Por exemplo,
- 🫡 (UTF-8 = 1FAE1) - 🥹 (UTF-8 = 1F979) - 😎 (UTF-8 = 1F60E) - 😈 (UTF-8 = 1F608)
