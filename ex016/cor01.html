Como colocar Cores no CSS

Representando Cores pelo Nome

Basta colocar como valor do parêmetro o nome da cor em inglês

Representando Cores por Código Hexadecimal

Basta colocar o código hexadecimal na forma #XXXXXX (6 dígitos após o #). Os valores simbolizam a intensidade de cada componente RGB. A intensidade varia de 0 a 256, ou 00 à FF em Hexadecimal. Para cada cor, na ordem R-G-B, temos 2 dígitos hexadecimais representando sua intensidade.

Representando Cores por Código RGB

Nesse caso o valor é dado usando a Função CSS " rgb (RED, GREEN, BLUE) ". No lugar de RED, GREEN e BLUE vc deve colocar um valor decimal equivalente à intensidade de cada cor (indo de 0 à 256)

Representando Cores por Código HSL

HSL é uma sigla que representa três componentes essenciais de uma cora - Matiz(Tom), Saturação e Luminosidade. É usado com a função CSS " hsl (HUE, SATURATION, LIGHTNESS) " onde HUE é um numero de 0 a 256 e representa o tom da cor (verde, amarelo, azul, etc). SATURATION é uma porcentagem de 0% à 100% e representa a saturação da cor (o que pura, intensa, ela é) e LIGHTNESS também é um porcentagem de 0% à 100%, representando o brilho da cor (mais escura ou mais clara, respectivamente).

Usando o VS Code ao nosso favor

Basta passar o mouse por cima do texto da cor que o VS code abre um gerenciador de cores completo. Clique na parte superior (onde está escrito RGB) que ele muda a representação para diferentes sistemas - RGB, Hexadecimal, HSL, etc -
Além disso, também é possível selecionar a transparência
