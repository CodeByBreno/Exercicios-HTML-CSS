Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>
Essa é uma tag não semântica, ou seja, apenas modifiquei a forma do texto,
sem atribuir um significado à isso. Logo, é uma tag indesejada no HTML

O que seria ide al então? Bom, geralmente o negrito é apenas um
texto em destaque, portanto a tag HTML deve ilustrar
esse significado, o de destacar texto. Para isso, existe a tag <strong>

Para itálico, existe a tag não semântica <i>

mas podemos fazer isso usando a tag semântica de enfâse : <em>

Texto Grande e Pequeno

Podemos tornar um texto grande ou pequeno usando as tags <big> e <small>
A tag "big" se tornou obsoleta, mas a "small" não

Uma explicação para isso é que "letras miúdas" ou "pequenas"
possuem significado semântico em uma série de contextos, como
contratos, notas de rodapé, legendas, e etc
Porém um texto grande, a principio, não. Se for um título, use
as tags <h>, se for um destaque/negrito, a tag <strong>, etc

Texto Sobrescrito

É aquele texto voando. Tipo a potência do x em x2+3

Para isso, use a tag <sup>

Texto Subescrito

É o texto embaixo, tipo o 2 de H2O

Para isso, use a tag <sub>
