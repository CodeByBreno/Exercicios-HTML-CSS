Hora de aprender sobre Parágrafos e Quebras!

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo entre um par de tags <p> e <p>.

Ou seja, o que existe entre tags <p> sempre será apresentado em uma única linha

Para dar uma quebra no texto, é preciso usar a tag sem fechamento <br>
Tipo assim

Mas porquê isso?

A razão para isso é que o HTML é uma linguagem de marcação, ou seja, seu objetivo é marcar os
componentes da página, estabelecendo a organização que terão entre sí. Dessa maneira, ao colocar
algo entre as tags <p>, a linguagem apenas entenderá que existe um texto a ser apresentado
sem levar em consideração sua formatação. Para especificar a formatação, é necessário usar outras
tags, ou estilos em CSS
